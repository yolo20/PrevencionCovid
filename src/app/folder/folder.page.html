Inicio Sintomas Como lavarse las manos correctamente

Paso 1: Mojar las manos con agua corriente segura

Paso 2: Aplicar suficiente jabón para cubrir las manos húmedas.

Paso 3: Frotar todas las superficies de las manos–incluyendo el dorso de las manos, entre los dedos y bajo las uñas–durante al menos 20 segundos.

Paso 4: Aclarar las manos a fondo con agua corriente

Paso 5: Secarse las manos con un paño limpio y seco, una toalla de un solo uso o un secador de manos, según sea el caso
